# 《美丽新世界》角色三视图提示词 - 标准模板

> 更新时间：2026-03-23

---

## 标准模板要素

| 要素 | 说明 |
|------|------|
| **角度** | 正视图 / 侧视图 / 后视图 |
| **背景** | 纯白色背景 (white background) |
| **范围** | 完整全身像，从头到脚全部身体 |
| **姿势** | 姿势自然，自然站姿 |
| **技术** | 电影感，8k，高细节，锐利焦点 |

---

## 标准格式

```
[年龄]岁[性别][职业][角度]视角，[完整全身像]，
[体型描述]，[脸部细节]，[眼镜/配饰]，
[服装描述]，[标志性特征]，[自然姿势]，
纯白色背景，电影感，8k，高细节，锐利焦点
```

### 负面词

```
变形，丑，模糊，扭曲，低质量，截图，
动漫，插画，素描，艺术化处理，截断，
头部缺失，脚部缺失，身体不完整
```
