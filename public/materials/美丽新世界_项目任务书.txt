# 《美丽新世界》AI视频预告片 — 项目任务书

> 创建时间：2026-03-23
> 状态：设计暂定，待启动

---

## 一、项目概述

- **作品名**：美丽新世界
- **类型**：科幻悬疑AI预告片
- **时长**：约50秒
- **核心概念**：未来能量被抽干的末日世界，"新世界"装置抽取本世界未来生态势能，四人在真相与谎言中交织
- **参考风格**：维伦纽瓦 +《湮灭》+ 真探S1

---

## 二、视觉风格（暂定）

### 调色板

| 状态 | 主色 | 辅色 | 情绪 |
|------|------|------|------|
| 正常世界 | 冷灰蓝 #4A5568 | 暗青 #2D3748 | 压抑、沉闷 |
| 新世界装置区 | 病态紫 #6B46C1 | 脉冲青 #00D9FF | 诡异、能量涌动 |
| 能量残影/坍缩 | 苍白绿白 #E8FFF0 | 血红 #E53E3E | 恐怖、不可名状 |
| 记忆/过去 | 暖琥珀 #F6AD55 | 褐色 #C4A77D | 温情但即将破碎 |

### 光影逻辑

- **主世界**：散射光，自然但稀薄，像阴天——无力的光
- **新世界装置**：体积光 + 边缘光，紫蓝脉冲，像什么东西在呼吸
- **能量残影**：廉价的白绿荧光，从地面/裂缝渗出，无来源方向
- **过去记忆**：暖色逆光，轮廓化，有诗意但克制

### 大师参考

| 导演/作品 | 拿来什么 |
|-----------|----------|
| 维伦纽瓦《降临》 | 冷灰蓝调/散射光/压迫感 |
| 维伦纽瓦《银翼杀手2049》 | 巨物感/霓虹紫/废墟美学 |
| 《湮灭》 | 能量异变/诡异生态/荧光白绿 |
| 真探S1 | 压抑调查感/冷青色调/人物特写 |

## 三、人物设计（暂定）

### 角色1：林深
- **身份**：28岁，观测院数据整理者
- **黄金锚点**：左颧骨旧伤疤；黑框眼镜（总歪）
- **服装**：深灰蓝工作服 #2C3E50
- **光影**：冷青屏幕光 + 散射日光

### 角色2：苏晚
- **身份**：26岁，文献馆纪元文字破译者
- **黄金锚点**：右手无名指墨水渍；左手腕红绳
- **服装**：米白考古服 #F5E6D3
- **光影**：暖琥珀台灯 + 冷蓝环境光

### 角色3：陈敬言
- **身份**：40岁，管理局行动司司长
- **黄金锚点**：右手臂"新世界"纹路疤痕
- **服装**：纯黑制服 #1A1A1A，暗红肩章 #8B0000
- **光影**：顶光+轮廓光，干净冷冽

### 角色4：孟清和
- **身份**：65岁+隐居老学者
- **黄金锚点**：左眼下方老年斑；左手少一小指
- **服装**：深褐旧袍 #3E2723
- **光影**：低光源蜡烛，暖褐为主

## 五、预告片结构（50秒，暂定）

```
[0-5s]   冷开场：废墟大全景 + 低沉环境音 + 字幕"文明之后"
[5-15s]  新世界装置：巨物特写 + 脉冲光 + 字幕"他们以为是答案"
[15-25s] 四人快速亮相（每人一镜）
[25-35s] 坍缩降临：残影涌动 + 世界扭曲 + 剪辑加速
[35-42s] 核心矛盾：两条线索交叉剪辑 + 字幕"每一步都被算计"
[42-50s] 终极画面：新世界全貌 vs 四人对峙 + 血红色标题
```

## 六、工作流程

```
Phase 1 — 生图（即梦AI）
├── 10场景独立生图
├── 每场景多版本备选
└── 固定满意图的Seed

Phase 2 — 图生视频（即梦AI）
├── 每镜5-10秒动态
├── 运镜+动作提示词
└── 保持首帧一致性

Phase 3 — 剪辑（剪映Pro）
├── 序列组装
├── 调色（统一冷蓝紫基调）
├── BGM（压抑/悬疑/科幻）
├── 字幕/标题
└── 输出成片
```
