## 一、视觉风格定调

**核心参考：维伦纽瓦 + 一点点《湮灭》**

> "死寂本身是一种压迫。"

### 调色板

| 状态 | 主色 | 辅色 | 情绪 |
|------|------|------|------|
| 正常世界 | 冷灰蓝 #4A5568 | 暗青 #2D3748 | 压抑、沉闷 |
| 新世界装置区 | 病态紫 #6B46C1 | 脉冲青 #00D9FF | 诡异、能量涌动 |
| 能量残影/坍缩 | 苍白绿白 #E8FFF0 | 血红 #E53E3E | 恐怖、不可名状 |
| 记忆/过去 | 暖琥珀 #F6AD55 | 褐色 #C4A77D | 温情但即将破碎 |

### 光影逻辑

- **主世界**：散射光，自然但稀薄，像阴天——无力的光
- **新世界装置**：体积光 + 边缘光，紫蓝脉冲，像什么东西在呼吸
- **能量残影**：廉价的白绿荧光，从地面/裂缝渗出，无来源方向
- **过去记忆**：暖色逆光，轮廓化，有诗意但克制

### 场景视觉关键词

```
荒原/废墟/观测院/文献馆禁地
↓
锈色金属 + 覆土 + 玻璃碎裂 + 植被入侵
↓
新世界装置 = 巨大环形结构 + 脉冲光纹 + 悬停的碎屑
↓
能量残影 = 人形轮廓抖动/透明度闪烁/无声嘶吼
```

## 七、预告片结构（50秒）

```
[0-5s]   冷开场：废墟大全景 + 低沉环境音 + 字幕"文明之后"
[5-15s]  新世界装置：巨物特写 + 脉冲光 + 字幕"他们以为是答案"
[15-25s] 四人快速亮相（每人一镜）：林深压抑/苏晚警觉/陈敬言冷峻/孟清和深沉
[25-35s] 坍缩降临：残影涌动 + 世界扭曲 + 剪辑加速
[35-42s] 核心矛盾：两条线索交叉剪辑 + 字幕"每一步都被算计"
[42-50s] 终极画面：新世界全貌 vs 四人对峙 + 血红色标题"美丽新世界"
```
