# 《美丽新世界》AI视频预告片 — 角色设计丰富版

> 更新时间：2026-03-23
> 设计理念：每个角色都是一个"世界"，通过细节讲述故事

---

## 角色深化核心原则

### 视觉叙事三层次

| 层次 | 内容 | 作用 |
|------|------|------|
| **一眼层** | 轮廓/体型/姿态/服装主色 | 5米外可辨认 |
| **二手层** | 面部特征/黄金锚点/配饰 | 情感第一印 |
| **三读层** | 道具/磨损/表情微变化/光影 | 深层次信息 |

### 角色弧光设计

每个角色都要回答：**"他们失去了什么？他们想找回什么？"**

## 角色1：林深

### 身份档案

| 属性 | 内容 |
|------|------|
| 年龄/身份 | 28岁，观测院数据整理者（三等研究员） |
| 编号 | OBS-2047-L |
| 入职年限 | 7年（父母原为观测院核心成员） |
| 现状 | 困在系统底层，知道太多不该知道的事 |

### 身体语言设计

```
站姿：重心偏左（习惯性防御）
走路：脚步轻快但刻意放慢（不想引人注意）
伏案：左肘撑桌，右手操作（保护左侧颧骨旧伤）
戴眼镜：频繁推眼镜，但只用右手（左手从不碰眼镜）
看屏幕：眯眼（显示他的眼睛其实对蓝光敏感）
```

### 衣着材质细节

| 部位 | 描述 |
|------|------|
| 工作服 | 深灰蓝 #2C3E50，涤纶混纺，肘部磨薄 |
| 领口 | 扣子只系到第二颗（不修边幅） |
| 袖口 | 磨损发白，左袖有咖啡渍 |
| 工牌 | 挂绳断过，用红线重新系上 |
| 裤子 | 黑色工装裤，膝盖处有褶皱（常跪姿操作设备） |
| 鞋子 | 旧款工程靴，鞋带换过三次 |

### 黄金锚点详解

**锚点1：左侧颧骨旧伤疤**
- 形状：不规则细长，1.5cm
- 故事：12岁时意外，但他在叙述中总是含糊其辞
- 视觉作用：他的手下意识用左手去碰触
- 光影：伤疤在冷青光下会形成微型阴影

**锚点2：歪斜的黑框眼镜**
- 左镜腿比右镜腿短2mm
- 右鼻托磨平了，左鼻托正常
- 左镜片有细微划痕（他一直没换）
- 眼镜总往右歪 ≈ 他的世界总在"失衡"

### 道具深度档案

**父母遗留的数据板**
- 型号：OBS-初代数据板，已停产20年
- 外观：边角磕碰，屏幕有暗斑，但贴膜完整
- 贴膜下：刻着一行字 "真相不会让你自由，但会让你清醒"
- 他用这块数据板的理由："习惯"

**左手腕监测手环**
- 款式：旧款生物监测手环（第4代，已被强制升级）
- 问题：心率监测不准，但他拒绝换新的
- 表带：表带内侧有轻微灼烧痕迹（某次事故的证据？）

### 表情情绪层次

| 情绪 | 面部细节 |
|------|----------|
| **压抑** | 眉头紧锁，嘴角形成"m"形纹路 |
| **警觉** | 眨眼频率加快，瞳孔轻微收缩 |
| **疲惫** | 眼窝阴影加深，上眼睑下垂 |
| **恐惧** | 下唇被牙齿轻咬，喉结滚动 |
| **决心** | 下巴微微上抬，眼眉距缩短 |

### 林深专属光影

```
主光：头顶散射冷青光（观测院标准照明）
辅光：屏幕冷蓝光（面部下方补光）
轮廓：稀薄的紫色边缘光（暗示"新世界装置"的影响）
地面反光：蓝色（工业地板漆）
```

### 人物关系网

| 关系 | 对象 | 性质 |
|------|------|------|
| 父母 | 已故观测院研究员 | 真相的起点 |
| 苏晚 | 文献馆 | 潜在盟友，但互不信任 |
| 陈敬言 | 管理局 | 被监视对象，恐惧与愤怒 |
| 孟清和 | 隐居学者 | 他在寻找的人 |

### 隐藏细节（镜头语言）

```
当他发现异常数据时：
- 手指会停止操作，悬停3秒
- 喉结会上下滚动一次
- 然后迅速切换屏幕（习惯性掩盖）

当他想起父母时：
- 会无意识地摸数据板边缘
- 眼镜歪了也不推正（那一刻他不想"看清"）
```

| 口袋 | 左侧口袋塞满碎片样本 |

### 黄金锚点详解

**锚点1：右手无名指墨水渍**
- 颜色：蓝黑墨水，渗入指纹纹路
- 形状：不规则晕开，约1cm²
- 故事：破译第一份纪元文献那天留下，洗不掉
- 象征：她与"过去"的连接

**锚点2：左手腕红绳**
- 材质：褪色红绳，编织紧密
- 长度：绕腕三圈，末端打结
- 故事：外婆的遗物，来自"那个世界"前
- 她戴在左手腕内侧（离心脏更近）

### 道具深度档案

**老旧放大镜**
- 框材质：氧化黄铜，边缘磨得发亮
- 镜片：有些微划痕，放大倍数3x
- 镜柄：有浅浅的指甲掐痕（她思考时的习惯）

**腰间配饰**
- 放大镜用细皮绳挂在腰带扣上
- 腰间还有：小铲刀、样本袋、纪元文字对照手册

### 表情情绪层次

|------|----------|
| **专注** | 眉头平展，眨眼间隔拉长至8秒 |
| **警觉** | 眨眼频率加快，鼻翼轻微扩张 |
| **发现** | 瞳孔放大，呼吸暂停1秒 |
| **恐惧** | 牙关紧咬，下颌肌肉紧绷 |
| **温柔** | 眼轮匝肌放松，视线变软 |

### 苏晚专属光影

```

轮廓：书籍轮廓形成的剪影边缘
情绪光：破译成功时，台灯会"感觉"更亮

| 陈敬言 | 管理局 | 直接对立，有历史 |

```

- 轻声念出破译内容（不自觉）
